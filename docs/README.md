# Simple OS 문서

이 디렉토리에는 Simple OS 프로젝트의 문서가 포함되어 있습니다.

## 문서 목록

### 핵심 문서

- **[architecture.md](architecture.md)**: 시스템 아키텍처 및 설계 문서
  - 프로젝트 목표 및 요구사항
  - 시스템 아키텍처 구조
  - 기술 스택 결정
  - 타겟 하드웨어
  - 설계 원칙

- **[requirements.md](requirements.md)**: 프로젝트 요구사항 문서
  - 기능적 요구사항
  - 비기능적 요구사항
  - 우선순위
  - 성공 기준
  - 리스크 관리

### 추가 문서 (향후 추가 예정)

- `api.md`: API 문서
- `developer-guide.md`: 개발자 가이드
- `user-manual.md`: 사용자 매뉴얼
- `adr/`: 아키텍처 결정 기록 (Architecture Decision Records)

## 문서 작성 가이드

- 모든 문서는 마크다운 형식으로 작성됩니다
- 코드 예제는 가능한 한 실행 가능하도록 작성합니다
- 아키텍처 변경사항은 `architecture.md`에 반영됩니다
- 중요한 설계 결정은 ADR로 기록됩니다
